# West Flora Export — Wholesale Flower Ordering App

Mobile app for wholesale flower ordering with product catalog, categories, client profiles, city selection and backend integration.

The project was built for a wholesale flower export business and focused on simplifying product ordering and customer interaction.

## Technical challenges

### Wholesale catalog structure

The app required a catalog structure suitable for wholesale ordering, not just simple retail browsing.

I designed the product and category logic so users could browse items and interact with the catalog in a practical mobile flow.

### Supabase backend

The backend was built with Supabase and PostgreSQL, including database structure and app-side integration through FlutterFlow.

### 1C integration planning

The project required preparing the app structure for future data exchange with 1C, including product and order-related data.
